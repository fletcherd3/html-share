research brief · 21 july 2026

free, open-source macOS dictation that doesn’t rent you the LLM

Alternatives to Superwhisper and Wispr Flow for system-wide voice dictation: local transcription, personal vocabulary, optional transcript cleanup with a key or model you control, and ideally a live transcript.

short answer

Start with TypeWhisper. It is the strongest all-round match: polished native UI, local models, live partial transcription, a serious dictionary/corrections system, and cleanup workflows using local models or your own API keys. Try Looped Whisper second if you prefer a smaller, more focused app with the same essential ingredients.

the shortlist

best overall

TypeWhisper

A close practical replacement for Superwhisper with the pricing model fixed: use local ASR and local cleanup, or route cloud cleanup through your own provider key.

  • System-wide push-to-talk, toggle, or hybrid hotkeys; auto-paste; a notch/overlay/minimal UI.
  • WhisperKit provides a partial live transcript as you speak.
  • Terms improve recognition; corrections repair common mistakes; it can learn high-confidence corrections and load term packs.
  • Per-app / per-site workflows can change engine, language, prompt, cleanup style or submission behaviour.
  • macOS 14+; universal binary, Apple Silicon recommended. Install via DMG or brew install --cask typewhisper/tap/typewhisper.
best focused option

Looped Whisper

The cleanest exact match on paper: local Whisper, real-time captions, a vocabulary feature, and cleanup through an API endpoint you control.

  • Apple Silicon, macOS 14+; Core ML / WhisperKit local transcription.
  • Anthropic and OpenAI-compatible cleanup; keys live in macOS Keychain.
  • Notarized DMG and Homebrew cask available.
  • Main trade-off: it is new, so independent testing and community feedback are still thin.
best broader OSS ecosystem

Handy

A well-known local-first project for macOS, Windows and Linux. Strong if portability and an entirely-local Ollama cleanup route matter more than Mac-native polish.

  • Local Whisper/Parakeet, local VAD, dictionary, and system-wide dictation.
  • The companion project can chain Gemini API, Claude, or Ollama cleanup.
  • Install on macOS with brew install --cask handy.
  • HN feedback praises Parakeet performance, while noting some advanced settings are less discoverable.
best simple local-only app

Pindrop

A native, no-subscription local dictation app with vocabulary and replacements. Great if raw transcription plus text replacements is enough.

  • WhisperKit/Core ML and Parakeet, with local streaming support.
  • Custom vocabulary and replacements; global shortcut and automatic insertion.
  • MIT licence; macOS 14+, Apple Silicon recommended.
  • It does not currently offer the rich BYOK LLM post-processing that makes TypeWhisper or Looped Whisper more compelling.

other good apps that miss the strict brief

VoiceInk: strong UX reputation and local models/dictionary, but the prebuilt app is trial/licensed; building from GPL source is free.
Local Whisper: excellent fully-local Parakeet/Qwen/Ollama/LM Studio path, but licensed PolyForm Noncommercial — source-available, not OSI open source.
MacWhisper: probably the established polished control option, with local models and BYOK/local AI processing, but it is not open source and premium features are paid.

what people seem to think

TypeWhisper has the clearest evidence of active momentum: roughly 1.6k GitHub stars, 30 contributors, frequent releases, and several independent write-ups calling out its live overlay, dictionary, provider choice and clean Mac UI. VoiceInk is the recurring community pick for polish, but fails the no-payment requirement. Handy has positive reports on speed and local Parakeet performance, though less admiration for its discoverability. The newer Looped Whisper and Pindrop have the right feature sets, but insufficient independent sentiment to treat as proven.

a sensible two-app test

  1. Install TypeWhisper and choose WhisperKit for live preview, or Parakeet for speed.
  2. Build a tiny dictionary: your name, company/product names, people you message often, technical words, and commonly mangled terms.
  3. Set up two cleanup workflows: one fully local through a local model, another using an API key you own for comparison.
  4. Test it against Looped Whisper with the same 5–10 real dictations: a message, an email, technical jargon, a noisy room, and a long thought.
  5. Keep the one that gets out of your way. UI latency and edit rate matter more than benchmark claims.
